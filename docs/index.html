Programmatic Usage

You can also use the library programmatically in your Python code:

from financial_analysis.filing_parser import FilingParser
from financial_analysis.config import SEC_API_KEY

# Initialize parser
parser = FilingParser("aapl", 2024, SEC_API_KEY)

# Access income data
revenue = parser.income.Revenue["net"]
net_income = parser.income.NetIncome
gross_profit = parser.income.GrossProfit

# Calculate metrics
profit_margin = (net_income / revenue) * 100
print(f"Profit Margin: {profit_margin:.2f}%")

# Export to JSON
print(parser.income.model_dump_json(indent=4))

Data Flow

1. Query Phase: Search SEC EDGAR database for latest 10-K filing

2. Fetch Phase: Download XBRL filing data via sec-api

3. Parse Phase: Convert XBRL to JSON and identify financial statement keys

4. Extract Phase: Apply keyword matching to extract relevant metrics

5. Validate Phase: Verify data integrity with Pydantic models

6. Calculate Phase: Compute derived metrics (Gross Profit, EBITDA, Net Income)

7. Output Phase: Return structured financial data and cache locally
